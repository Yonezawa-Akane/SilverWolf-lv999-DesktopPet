============================================
   银狼桌宠 v2.1.2 · 快速开始
============================================

3 步上手：

【1】把这整个文件夹复制到 C 盘以外的位置
     例如 D:\SilverWolfPet\
     （C:\Program Files 需要管理员权限，不推荐）

【2】双击  SilverWolfPet.exe
     首次启动会问你"要不要在桌面放快捷方式？"
     选好之后下次双击桌面 🐺 图标就能启动

【3】配置 API Key
     在 sidebar 顶部 ⚙ 按钮里粘贴
     形如 sk-ant-api03-XXXXXXXXX...


============================================
   如何获取 API Key
============================================

打开浏览器：
  https://console.anthropic.com/

   1. 注册账号（邮箱即可）
   2. 进入左侧菜单 "API Keys"
   3. 点 "Create Key" 创建一个新 key
   4. 复制以 sk-ant- 开头的字符串

注意：API 是按用量付费，需要绑定信用卡或先充值。
价格非常便宜——日常使用每月 5-20 美元足够。
新用户首次充值有 $5 免费额度可以试用。


============================================
   首次启动 SmartScreen 警告
============================================

Windows 可能会提示：
  "Windows 已保护你的电脑"
  "Microsoft Defender SmartScreen 阻止了启动..."

正常现象。本程序未做代码签名（个人开发免不了），
但代码完全开源透明、不收集任何数据。

操作：
  → 点击  "更多信息"
  → 出现  "仍要运行"  按钮
  → 点击它

下次启动就不会再问。

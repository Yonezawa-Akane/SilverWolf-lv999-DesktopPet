# 银狼桌宠 · 使用说明书

> 本文是 silver-wolf-pet 桌面 AI 伴侣的完整文档。从安装到日常使用，到常见问题排查。
>
> 当前版本：**v2.2.2** · 最新变更见项目根目录 `CHANGELOG.md`

---

## 1. 这是什么

silver-wolf-pet 是一个 Windows 桌面 AI 伴侣应用，主角色是崩坏：星穹铁道的银狼。她不是一个普通的聊天机器人——是一个有完整人设、跨 session 长期记忆、屏幕感知能力的桌面拍档：

- **能聊天**：用银狼的语气，记得你和她聊过什么（跨重启不丢）
- **能开应用、网址、文件夹**：自然语言指令直接执行
- **能分析屏幕**：按 `Ctrl+Shift+\` 截图，5 种模式秒读屏内容
- **能陪你工作**：番茄钟节奏，工作时安静、阶段切换会出声
- **能管理任务**：sidebar 任务 tab 自己加，AI 银狼也能帮你勾

## 2. 系统要求

- Windows 10 64位 或 Windows 11
- 至少 200 MB 磁盘空间
- 互联网连接（调用 Claude API 时需要）
- Anthropic API Key（自行注册，价格见下文）

## 3.0 30 秒看完就上手（新手必读）

> 假设你刚从 Releases 页面下完 `SilverWolfPet-v2.2.x.zip`，电脑用着是「日常会双击图标」的水平。
> 严格按下面 5 步走，10 分钟之内就能跟银狼聊天了。

### 步骤 1 ─ 📦 解压

- 找到下载好的 `SilverWolfPet-v2.2.x.zip`，**右键** → "解压到当前文件夹" 或 "解压到 SilverWolfPet/"
- 把整个解压出来的文件夹拖到 **D 盘根目录**，最终路径形如 `D:\SilverWolfPet\`
- ⚠ **不要放** `C:\Program Files` 或 `C:\Windows`，这些位置需要管理员权限会装出毛病
- 拖错了？没关系，删掉文件夹重来即可。本应用不污染系统注册表，所有数据在 `%APPDATA%\silver-wolf-pet\`（卸载时连这个目录一起删就清干净）

> 卡这一步？看 §3.3。

### 步骤 2 ─ 🛠️ 配置语音环境（仅首次）

- 进解压出来的文件夹，找到名叫 **「① 配置环境.bat」** 的文件
- **双击它**，会弹出一个**黑色命令行窗口**（这是正常的，别关）
- 看到下面任意一行就算成功，按任意键关闭：
  - `[OK] VC++ 运行库已安装` （之前装过）
  - `[OK] VC++ 运行库安装成功` （这次刚装好）
  - `[OK] VC++ 运行库已是最新版本` （已经是新版了）
- 全程约 30 秒，**仅首次需要**，下次启动跳过这步

> 卡这一步？.bat 闪退或报红？看 §8.14。诊断日志在 `%APPDATA%\silver-wolf-pet\logs\voice-init.log`。

### 步骤 3 ─ 🚀 启动应用

- 双击 **「② 启动银狼.bat」**（不是 `SilverWolfPet.exe`，虽然两者效果一样，但 .bat 入口将来可能多一些环境检查）
- 第一次启动可能蹦出**蓝色全屏窗口**：
  > **Windows 已保护你的电脑**
  > Microsoft Defender SmartScreen 阻止启动了无法识别的应用...
- 操作（看清楚两个按钮在哪）：
  1. 点窗口左下角的蓝色小字 **「更多信息」**（注意是小字，不是右下角的"不运行"按钮）
  2. 点开后下方会**冒出一个新按钮 「仍要运行」**，点它
- 之后每次启动都不会再问

> 详细见 §3.4。点完仍要运行还是不开 / 没反应？看 §8.2，并尝试重跑一次 ① 配置环境.bat。

### 步骤 4 ─ 🔑 注册 Anthropic 账号 + 拿 API Key

银狼自己不会说话——她要靠 **Anthropic API**（提供 Claude 大模型的服务）才能产生回复。要用她，**必须**有一个 API Key。

操作：

1. 浏览器打开 **https://console.anthropic.com/**
2. 点 **「Sign Up」** 注册账号，邮箱 + 手机号
   - ⚠ **中国大陆用户**：境外手机号 / Google Voice / 朋友的境外号都行；境内 +86 号没法验证
   - ⚠ 直连这个网址可能直接被拒，需要 VPN / 代理（详见 §3.2）
3. 注册成功后默认有 **5 美元试用额度**（够聊几千次）
4. 进控制台后，左侧菜单 → **「API Keys」** → **「Create Key」** → 名字随便（比如 `silver-wolf`）→ 点 **「Copy」** 复制
5. 这串以 **`sk-ant-api03-`** 开头的字符串就是你的 key，先粘到记事本备用

> 卡这一步？见 §3.2「Anthropic 注册踩坑」。

### 步骤 5 ─ 💬 把 API Key 粘到银狼里

- 启动后银狼出现在桌面（粉发小人，约 1/4 屏幕高度）
- 把鼠标移到银狼身上，**右侧紫色聊天面板**（sidebar）会自动浮出
- sidebar 顶部右上角有个齿轮 **`⚙`** 按钮 → 点它打开设置
- 在 `Anthropic API Key` 输入框里**粘贴**你刚刚复制的 `sk-ant-api03-...` 字符串
- 点 **`▶ SAVE`** 保存，设置面板自动关闭

> 卡这一步？粘了 key 还报红色 ERROR: Request Not Allowed？看 §8.15。

### ✓ 完成

- 在 sidebar 底部输入框打字 + 回车 → 银狼回复
- **首次回复会慢 2-5 秒**（连 Anthropic 服务器中），之后正常
- 想用语音？按 `Ctrl+Alt+V`（详见 §3.1）
- 想截图分析？按 `Ctrl+Shift+\`（详见 §3.1）
- 想看更多功能：跳到 §3.1
- 想了解每个步骤的细节：从 §3.3 起一路读下去

## 3.1 5 分钟掌握核心功能

### 💬 聊天

- 鼠标悬停银狼 → sidebar 浮出 → 底部输入框打字 + 回车
- 银狼会**记住你说过什么**——跨重启不丢，存在 `%APPDATA%\silver-wolf-pet\state.json`
- 想清空历史：sidebar ⚙ 设置面板 → 红色「⌧ 清空对话历史」按钮
- 详见 §4.2 / §4.6

### 🎙️ 语音输入（PTT）

- 默认热键：**`Ctrl + Alt + V`**
- 按一下 → 银狼气泡冒出「🎙️ 听着呢...」→ 直接说话
- 再按一下 → 自动识别成文字 → 塞进 sidebar 输入框 → 自动发送
- 中英粤日韩自动识别，**完全离线**，不联网
- 不想用？sidebar ⚙ 设置面板里改 voice_input_enabled
- 详见 §4.10。卡住了见 §8.14

### 📸 屏幕分析

- 默认热键：**`Ctrl + Shift + \`**（任何应用前都生效）
- 按一下 → 屏幕中央弹 5 模式面板：解释 / 翻译 / 找错 / OCR / 总结
- 按 **数字键 1-5** 直接选模式，**ESC** 取消
- 详见 §4.3

### 🍅 番茄钟

- 跟银狼说 `开始番茄钟` 或 `开 50 分钟番茄钟，写代码`
- 屏幕右上角浮出倒计时面板，工作时银狼**保持安静**；切休息时她会出声提醒
- 取消：面板上 ✕ STOP，或跟银狼说"取消番茄"
- 详见 §4.4

### 📌 任务清单

- sidebar 切到 📋 任务 tab → 输入框加任务 / 点 ☐ 勾选 / hover 任务条 → 右侧 ✕ 删除
- 也可以直接对银狼说 `加个任务：明天交作业`，她会帮你勾
- 跨日打开应用时，如果有未完成任务，银狼可能主动提一句
- 详见 §4.2 任务 tab 部分

### 🪄 改快捷键 / 改外观 / 改 UI 缩放

- sidebar **`⚙`** 按钮 → 滚动看
- 三个全局热键（PTT / 截屏 / 召回）都能录制式自定义：点 **「⏺ 录制」** → 按下新组合即生效
- UI 缩放四档（小 / 默认 / 大 / 特大）即点即生效，不用重启
- 详见 §4.10「自定义热键」+ §4.1 三个独立窗口

## 3.2 万一卡在哪一步？

按 §3.0 的步骤号对照查：

| 卡哪一步 | 现象 | 看哪 |
|---|---|---|
| 步骤 1 | 解压失败 / 路径乱码 | §3.3 |
| 步骤 2 | ① 配置环境.bat 闪退 / 报红 | §8.14 + 看 `%APPDATA%\silver-wolf-pet\logs\voice-init.log` |
| 步骤 3 | 蓝屏 SmartScreen 点了"仍要运行"还是不开 | §8.2 + 重跑 ① 配置环境.bat |
| 步骤 4 | Anthropic 注册不了 / 验证手机失败 | 见下方「Anthropic 注册踩坑」 |
| 步骤 5 | 粘了 key 还报 ERROR: Request Not Allowed | §8.15 |

### Anthropic 注册踩坑（中国大陆用户）

中国大陆 IP 直连 console.anthropic.com 经常被 Anthropic 风控直接拒绝，常见三类问题：

1. **打不开注册页面**
   - 现象：浏览器转圈 / 报 `connection refused` / 显示英文 `Forbidden`
   - 原因：你的出口 IP 在 Anthropic 黑名单地区
   - 解决：挂 **VPN / 代理**（确保代理覆盖整个浏览器，不是只系统代理）后重试

2. **验证手机不能用 +86 号**
   - 现象：注册页输入手机号后报 `We don't support phone numbers from your region`
   - 原因：Anthropic 不支持中国大陆手机号验证
   - 解决：用 **Google Voice 号码 / 境外朋友手机号 / 海外接码服务**（搜索引擎搜 "sms 接码" 找一个临时号）

3. **注册成功但 API 调用 403 Request Not Allowed**
   - 现象：sidebar 输入消息回车后，对话区出现红色 `Request Not Allowed`
   - 原因：你的**当前出口 IP** 在 Anthropic 黑名单。注册时用了代理，但日常没开 → 银狼的 API 调用走的是直连大陆 IP
   - 解决：**全程开代理**，并且代理必须覆盖 `api.anthropic.com`（不只代理浏览器）
   - 详细见 §8.15

## 3. 第一次启动

> ⓘ §3.0 已经把首次启动的 5 步走完了。下面是各步骤的展开说明 / 补充选项。

### 3.3 解压

把 zip 解压到你想要的位置。**不要放 `C:\Program Files`**（需要管理员权限），推荐：

- `D:\SilverWolfPet\`
- `D:\Tools\SilverWolfPet\`
- `C:\Users\<你的用户名>\Documents\SilverWolfPet\`

### 3.4 处理 Windows 安全提示

第一次双击 `SilverWolfPet.exe` 时，可能会跳出蓝色窗口：

> **Windows 已保护你的电脑**
> Microsoft Defender SmartScreen 阻止启动了无法识别的应用...

正常现象——本应用未做代码签名（个人开发免不了），但代码完全开源透明。

**操作**：

1. 点击窗口里的 **更多信息**
2. 会出现 **仍要运行** 按钮
3. 点它

下次启动就不会再问。

### 3.5 桌面快捷方式（首次询问）

启动后约 1.5 秒，会弹出一个 Windows 原生对话框：

> **要不要在桌面放个快捷方式？**
> 方便下次直接双击启动。可以随时在 Launcher 右键菜单 / 设置面板里改。

- 选 **好，放一个** → 桌面立即出现 🐺 SilverWolfPet 图标
- 选 **不用** → 不创建

**这个弹窗只问一次**。后续想改主意有两个入口：

- **Launcher 右键菜单** → `+ 创建桌面快捷方式` / `✓ 移除桌面快捷方式`
- **sidebar 设置面板**（⚙ 按钮）→ 同样的切换按钮

OneDrive 重定向桌面 / 中文 "桌面" 文件夹自动兼容。

### 3.6 配置 API Key

启动后你会看到：

- 屏幕**右下角**：紫色小六边形图标（launcher 召唤按钮）
- 屏幕某处：银狼小角色（会自己漂浮）
- launcher 图标自动点开 sidebar

sidebar 顶部三个按钮 `⚙ ─ ✕`，点 **⚙** 打开设置。

填入你的 Anthropic API Key（形如 `sk-ant-api03-XXXXXXXXX...`），点 SAVE。

> **API Key 只存在你本地的浏览器存储里，不会上传到任何服务器。**

### 3.7 怎么拿 API Key

1. 打开浏览器：[https://console.anthropic.com/](https://console.anthropic.com/)
2. 注册账号（邮箱）
3. 左侧菜单 → **API Keys**
4. 点 **Create Key**
5. 复制以 `sk-ant-` 开头的字符串

**关于费用**：

- Anthropic API 按用量付费，需要绑定信用卡或预充值
- 默认模型 Claude Sonnet 4.6 价格：$3/1M input / $15/1M output token
- **日常使用每月 5-20 美元足够**
- 一次普通聊天约 1-2 美分
- 一次截图分析（含图像识别）约 3-10 美分
- 新用户首次充值有 $5 免费额度

国内用户可能需要 VPN 或代理才能直连 anthropic.com（API 端点）。

## 4. 主要功能

### 4.1 三个独立窗口

应用启动后你会看到：

- **launcher 召唤图标**：右下角小六边形按钮
  - 点击：显示/隐藏 sidebar
  - 拖动：换位置（持久化到下次启动）
  - 右键：菜单（设置/重置位置/退出）
- **银狼角色**：屏幕上漂浮的 84×115 小人 + 旁边的气泡区
  - 鼠标靠近会跑（"别贴"）
  - 点击：展开 sidebar
  - 右键：菜单（DevTools/退出）
- **sidebar 主面板**：右侧紫色面板，三个 tab 切换

### 4.2 三个 sidebar tabs

#### 💬 对话 tab

跟银狼自然对话。完整人设、记忆、语气都在这里展开。

**支持的指令短语示例**：

| 你说 | 银狼做 |
|---|---|
| `打开 Chrome` | 启动 Chrome |
| `打开 微信` | 启动微信（含双语别名匹配） |
| `打开 Discord` | 启动 Discord |
| `在 b 站搜银狼` | 浏览器搜索 b 站 |
| `在谷歌搜 X` | 浏览器搜索 Google |
| `打开下载文件夹` | 打开 Downloads |
| `打开桌面` | 打开 Desktop |
| `开始番茄钟` | 启动 25 分钟工作 |
| `开 50 分钟番茄钟，写代码` | 自定义时长 + 标签 |
| `记一下 X` / `提醒我 Y` | 加任务到清单 |
| `X 完成了` / `勾掉 X` | 勾掉任务 |
| `让银狼飞过去点 X` | 视觉模式定位点击（低频） |
| `上次卡哪了` | 召回上次对话 |
| `看看屏幕里有什么` | 截图 + 分析 |

#### 📋 任务 tab

你的待办清单。

- **添加**：底部输入框输入文字 + Enter
- **勾选**：点 ☐ → 变 ☑（自动 strike-through 沉到底部）
- **删除**：hover 任务条 → 右侧出现 ✕ → 点删
- **AI 操作**：跟银狼说"记一下 X" / "X 完成了" 也能改

任务持久化到本地 `state.json`，跨重启不丢失。

任务数会显示在 tab 上的粉色 badge，即使在其他 tab 也能看到（`📋 任务 ②`）。

跨日打开应用时，如果有未完成任务，银狼可能会主动提一句。

#### 📖 说明 tab

应用的快速操作清单。当你忘记某个快捷键或银狼能干嘛时来这查（你正在看的这份文档是详细版）。

### 4.3 全局快捷键 — 截图智能助手

按 **`Ctrl + Shift + \`** （任何应用前都生效），屏幕中央出现 HUD 风格小面板：

5 个模式按钮：

1. **🔍 解释** — 银狼用她的语气解释屏幕内容是什么、用户在做什么
2. **🌐 翻译** — 自动检测：中文为主翻英文，其他语言（英/日/韩/法等）翻中文
3. **🐛 找错** — 看截图里的报错/异常，给修复建议
4. **📋 OCR** — 提取所有可见文字，**自动复制到剪贴板**
5. **📝 总结** — 长内容压成 3-5 句

按 **1-5 数字键** 直接选模式。**ESC** 取消。

**结果会显示**：
- sidebar 对话面板里有完整文字结果
- 银狼气泡同时冒出 "Aha 看一眼..." 之类的反馈

### 4.4 番茄钟

跟银狼说 `开始番茄钟` 或 `开 25 分钟番茄钟，写代码`：

- **屏幕右上角**浮出 240×180 番茄面板
- 大字倒计时 + phase 标签（🍅 WORK / ☕ BREAK）+ 任务标签 + 今日计数
- 默认 25 分钟工作 + 5 分钟休息（指令里可指定）
- 工作中银狼**保持安静**不打扰
- 阶段切换：
  - 工作完 → ☕ BREAK 模式 + 银狼气泡 + Windows 系统通知
  - 休息完 → 整轮结束 + 通知 + 面板自动隐藏
- 最后 1 分钟数字会**粉色脉冲**（紧迫感）
- 取消：面板上 **✕ STOP** 按钮，或跟银狼说"取消番茄"

每天的完成数自动重置（跨日检测），重启不丢。

### 4.5 银狼气泡

屏幕上银狼角色右侧会冒小气泡，触发时机：

- 截图模式触发时
- 番茄钟阶段切换
- 飞过去点击成功后
- 偶尔自言自语（quip）
- 鼠标靠近时

气泡用不同颜色表达情绪：

- **紫色** = default（正常说话）
- **青色** = processing（处理中）
- **绿色** = success（完成）
- **红色** = alert（警告）

### 4.6 长期记忆

银狼会**记住**：

- **事实**：你告诉她的偏好、当前项目、上下文（通过 `remember_about_user` 工具自动存）
- **每日小记**：她自己总结的对话节选（每天最多一条）
- **完整归档**：每次关闭程序时整段对话存档（保留 180 天）
- **未完成任务**：每次对话她都能看到清单

跨日（>8 小时）打开应用时，她会**主动**说一句基于记忆的开场白，比如：

> "Aha，回来了。上次卡 fly_and_click 那个修了吗？"

不会每次开都说，只在距离上次活跃超过阈值时触发。

### 4.7 应用启动 — 5 级策略

`打开 X` 这种指令背后实际有 5 级 fallback：

1. **Start Menu** 精确匹配
2. **Desktop .lnk** 精确匹配
3. **Desktop .lnk** 包含匹配
4. **Start Menu** 包含匹配（按 startsWith 优先）
5. **Uninstall 注册表** 扫描（捕获桌面/开始菜单都没条目的应用，如 Discord 用户级安装）

如果以上都未命中，会 fallback 到 Windows 搜索栏（`openViaSearch`）剪贴板粘贴 + 回车。

中文应用名走双语别名表（微信↔WeChat / QQ音乐↔QQMusic / 网易云↔CloudMusic / 钉钉↔DingTalk 等 16 条），命中率 ~95%。

### 4.8 视觉点击 (fly_and_click)

低频娱乐性能力。仅在你**明确说**"让银狼飞过去点 X" 时触发：

1. 银狼自动截图屏幕
2. AI 分析图像找到目标
3. 银狼角色物理飞过去
4. 在目标位置双击

**精度受限**：模型视觉估坐标有 ±5-15px 误差。图标密集时可能点错隔壁。日常启动应用还是 `打开 X` 更稳。

### 4.9 文件转换器（v2.1 新增）

**直接把文件拖到 sidebar 任何位置** —— 整个面板会变紫色虚线高亮，松手后弹出格式选择菜单。

**菜单结构**

- `// FORMAT //` 段：可转换的目标格式（紫色按钮，例如 `.JPG` `.PDF` `.HTML`）
- `// OPERATION //` 段：针对该文件类型的特殊操作（青色按钮，例如 `✂ 拆分（每页1个PDF）`、`↻ 旋转 90°`），仅 PDF 显示
- `[ESC] CANCEL` 关闭菜单

**支持的转换路径**

| 拖入文件 | 可转换至 |
|---|---|
| **图片** PNG / JPG / BMP / WebP | 图片互转、PDF |
| **PDF** | TXT / DOCX / PNG / JPG，✂ 拆分（每页1个PDF）、↻ 旋转 90° |
| **Word** DOCX | HTML / Markdown / TXT / PDF |
| **Markdown** MD | HTML / DOCX / PDF / TXT |
| **HTML** HTML / HTM | Markdown / DOCX / PDF / TXT |
| **TXT** | PDF / DOCX / HTML / Markdown |
| **Excel** XLSX / XLS | PDF / HTML / CSV |
| **CSV** | XLSX / HTML / PDF / TXT |

**输出位置 / 命名规则**

- 输出文件**回到原文件同目录**
- 同名加 `_1` `_2` 递增（例如：`a.pdf` 已存在 → 新文件 `a_1.pdf`）
- 多文件输出（PDF→图片、PDF 拆分）自动包**子文件夹**：
  - PDF → 图片：`<原名>_pages/page_01.png`、`page_02.png` ...
  - PDF 拆分：`<原名>_split/page_01.pdf`、`page_02.pdf` ...

**反馈提示**

- 转换中：银狼气泡 "稍等，编辑一下 .X 的以太层…"
- 完成：气泡 + sidebar 聊天区出 cyan 系统消息显示输出路径
- 失败 / 不支持：气泡 "这玩意我搞不动"

**已知限制**

- WebP 只能**作为源**（jimp 0.22 不内置 WebP 编码器，无 `→.WebP` 选项）
- PDF → DOCX 仅保留段落级文本（pdf-parse 限制，不还原表格 / 图片 / 排版）
- Excel → CSV 时仅导出第一个 sheet（CSV 单 sheet 格式所致），文件头会注释提示丢失了几个 sheet

### 4.10 语音对话 PTT

不想打字时，按热键说话直接发消息——本地 SenseVoice 模型当场识别成文字塞进输入框、发送，等同于打字回车后银狼正常回复。

**默认热键：`Ctrl + Alt + V`**（可在设置面板里改）。

**两种模式**（设置面板切换）：

- **📣 toggle（默认）** —— 按一次开麦、再按一次结束。基于 Electron 内置 `globalShortcut`，全 Windows 配置稳定。**推荐日常使用。**
- **🎙️ hold（按住说话）** —— Discord 风格，按住录音、松手即送。基于 `uiohook-napi` 低级钩子，部分 Windows 系统下钩子可能被 OS / 杀毒 / 输入法摘掉。万一卡住可点「⏹ 强制结束录音」复位。

**支持语种**：中、英、粤、日、韩 + 中英混说。完全离线，**零网络依赖**。

**典型流程**：

1. 按 `Ctrl+Alt+V` → 银狼气泡冒出「🎙️ 听着呢...」cyan 处理状态
2. 直接说，比如「帮我打开 Steam」
3. 再按 `Ctrl+Alt+V`（toggle）/ 松开热键（hold）→ 气泡转「💭 解析中...」
4. 识别完成 → sidebar 弹出「打开 Steam」的用户消息 + 银狼回应
5. 桌面 launcher 录音中变 cyan 高亮 + halo 加速，给余光提示

**自定义热键**：

- 设置面板 ⚙ → 滚到「全局热键 / Hotkeys」→ 点「⏺ 录制」→ 按下新组合即生效
- 三个全局热键（PTT / 截屏 / 召回）都能改
- 录制期间已有热键自动暂停，不会被吃键

**模型权重**：

- `model.int8.onnx` 约 234 MB，**不随 .exe 打包**（避免发行包过大）
- 首次使用前请按 [docs/voice-input-spec.md §8.1](./voice-input-spec.md) 单独下载到 `assets/models/sense-voice/` 目录
- 模型缺失时启动会弹气泡提示，PTT 暂时禁用，不影响其他功能

**已知限制**

- 短于 300ms 的录音视为误触，静默丢弃（防止 hold 模式手滑触发）
- 单次录音上限 30 秒（防止热键卡住无限录音）
- hotwords 热词表（项目专有词如「星核猎手」「GitHub」）已准备但当前 sherpa-onnx 对 SenseVoice **未开** modified_beam_search，自动回退无 hotwords。等上游开放后会自动启用，无需改配置
- hold 模式偶发的钩子失效场景：杀毒软件 / 第三方 IME / RDP / 部分全屏游戏。3 秒静默会自动安全释放，但当次录音丢失。**生产环境推荐用 toggle 默认。**

## 8. 常见问题

### 8.1 360 / 电脑管家拦截

部分国产安全软件会误报 Electron 应用。**允许放行**或加白名单。本应用代码开源透明，不会做任何恶意行为。

### 8.2 启动后银狼或 sidebar 不见

- 检查任务管理器是否有 `SilverWolfPet.exe` 进程
- v2.1.1 起，启动时会自动校验 launcher 位置——如果之前在外接显示器上、现在那个屏幕没了，launcher 会自动回到主屏右下角。运行时拔/插显示器也会自动救回所有窗口
- 极端情况下还是看不见，删除 `%APPDATA%\silver-wolf-pet\launcher_pos.json`，下次启动会强制重置到右下角

### 8.3 输入对话发不出去

排查顺序：

1. **API Key 是否填了**：sidebar ⚙ 检查
2. **Anthropic 账号是否有余额**：去 console.anthropic.com 看
3. **网络能否到达 anthropic.com**：浏览器试访问；大陆网络可能需 VPN
4. **API Key 是否被作废了**：在 console 看 key 状态

### 8.4 终端日志中文乱码

Windows cmd 默认中文代码页是 GBK，与 UTF-8 不兼容。

启动应用前先执行：

```cmd
chcp 65001
```

或者修改 `package.json` 的 start 脚本（仅影响 npm start 开发模式）。

打包后的 `.exe` 用户不会看到这些日志，所以这只是开发者调试时的提醒。

### 8.5 番茄钟阶段切换没系统通知

Windows 通知中心可能屏蔽了。打开 设置 → 系统 → 通知 → 检查应用通知是否开启。

### 8.6 Ctrl+Shift+\ 快捷键不生效

可能被其他应用占用。开发者模式下终端会打印：

```
[shortcut] 注册失败：可能被其他应用占用
```

修改 `state.json` 里的 `preferences.quick_insight_shortcut` 字段，常见替代值：

- `CommandOrControl+Alt+G`
- `CommandOrControl+Shift+I`
- `CommandOrControl+Shift+P`

修改后**完全关闭并重启**应用。

### 8.7 银狼飞错位置

屏幕分辨率/缩放变更后可能。重启应用让坐标重新计算。如果仍偏，是模型视觉精度上限，不是 bug。

### 8.8 想换银狼语气 / 高频词

编辑 `silver-wolf-skill-distilled.md`（项目内）或 sidebar.html 里的 `SW_PERSONA` 常量。详见开发者 README.md。

### 8.9 sidebar 太大太挤

sidebar 可拖动 + 可调整大小。鼠标拖动顶部 `// SIGNAL LINK //` 区域 = 移动；拖动边缘 = 缩放（min 220 / max 600 宽度）。

### 8.10 对话历史太长想清空

sidebar ⚙ 设置面板里有 `⌧ 清空对话历史` 按钮。**只清当前对话**，不影响归档和事实记忆。

要彻底清记忆：删除 `%APPDATA%\silver-wolf-pet\state.json`。

### 8.11 拖文件没反应 / 找不到拖放区

确认拖到的是 **sidebar 面板内任意位置**（不是 launcher 按钮或银狼角色）。拖入时整个面板边缘应该出现紫色虚线高亮、中间显示 `// DROP_FILE //`。如果完全没反应：

- 文件路径过长 / 包含特殊字符可能拿不到 `.path`，尝试把文件先复制到桌面再拖
- 多文件同时拖入会**逐个**弹菜单处理

### 8.12 文件转换报错 "不支持 .xxx → .yyy"

参考说明书 4.9 节的转换路径表，确认目标格式在支持范围内。常见误解：

- WebP 只能作为源，没有 `→.WebP` 选项
- PowerPoint (.pptx) / 旧 .doc 格式暂不支持
- 加密 PDF 无法处理

### 8.13 桌面快捷方式弹窗已经被我点了"不用"，怎么再创建

- 右键 launcher 圆形图标 → 点 `+ 创建桌面快捷方式`
- 或 sidebar ⚙ → 第二个 cyan 按钮 `+ 创建桌面快捷方式`

开发模式（`npm start` 而非打包后的 .exe）下按钮会 disabled 灰显，因为没法对开发模式的 electron.exe 创建有效快捷方式。

### 8.14 PTT 初始化失败 / 语音输入用不了

启动时气泡冒出 "PTT 暂时不能用" 类文案 → 按以下顺序排查：

1. **模型文件大小**：检查 `assets/models/sense-voice/model.int8.onnx` ≈ 234MB（实际约 234,344,448 字节）。如果只有几 KB 或几 MB，多半是下载未完成、被 Git LFS 占位、或被杀软隔离 → 删掉重新下载（见 [docs/voice-input-spec.md §8.1](voice-input-spec.md)）。
2. **VC++ 运行库**：v2.2.1+ 发行包已带 vc_redist.x64.exe，正常情况下首次跑配置环境.bat 就装好了。如果跳过了配置环境.bat 直接启动，或自动安装失败，手动跑解压目录里的 `redist\vc_redist.x64.exe` 即可。也可以从官网下载：[VC++ Redistributable 2019/2022 (x64)](https://aka.ms/vs/17/release/vc_redist.x64.exe)。sherpa-onnx 的原生 `.node` 模块依赖它；缺了表现为"包在但 require 抛错"。
3. **CPU 架构**：sherpa-onnx 目前只发 `win-x64` / `win-ia32` 子包。Windows ARM64（如 Surface Pro 9）暂时跑不了，需等 sherpa 出 ARM64 子包，或用 x64 版 Node 模拟。
4. **杀毒软件 / Windows Defender**：偶尔会把原生 `.node` 当病毒隔离。把 `node_modules\sherpa-onnx-*` 目录加白名单，或翻一下隔离区找回。
5. **热键被占用**：默认 PTT 是 `Ctrl+Alt+V`。如果被 OBS / IME / 其他全局钩子抢了，控制台会出现 `[shortcut] 注册失败` → 在 sidebar 设置里改成别的组合。

诊断日志位置：`%APPDATA%\silver-wolf-pet\logs\voice-init.log`（每次启动失败都会追加一条，含 stage / 模型大小 / 平台架构 / Electron 版本）。把这文件发给开发者能直接定位是哪一步挂了。

### 8.15 ERROR: Request Not Allowed / API 调用 403

sidebar 输入消息后，对话区出现红色 `Request Not Allowed` 或 `403`：

**最常见原因：当前出口 IP 在 Anthropic 黑名单地区**（典型：中国大陆直连）。Anthropic 服务端按 IP 风控，**和你的 API Key 是否有效无关**——一个有效 key 在禁地区一样会被拒。

**排查 + 修复**（按顺序）：

1. **确认网络能到 api.anthropic.com**
   - 浏览器访问 https://api.anthropic.com/v1/messages
   - 看到 JSON 响应（哪怕是 `{"error": ...}` 这种结构）就算通
   - 如果是 `connection refused` / 长时间转圈不出来 / SSL 错 → 出口 IP 不通，进第 2 步

2. **挂代理，并且代理必须覆盖整个系统的 https 流量**（不是只代理浏览器）
   - Clash / V2Ray 开 **TUN 模式** 或 **系统代理 → 全局**
   - 节点选 Anthropic 支持的地区（美 / 日 / 港 / 新加坡都行）
   - 代理只覆盖浏览器是不够的——银狼的 API 调用走的是 Electron 主进程 fetch，不读浏览器代理设置

3. **重启银狼桌宠** —— 代理改了之后必须重启让它重新建连

4. **检查 API Key 是否仍有效**
   - 登录 console.anthropic.com → 左侧 API Keys → 看 key 状态是否 `active`
   - 如果显示 `revoked` / `expired`，重新 Create Key 一次

5. **检查账户余额**
   - console 首页 → Credits → 至少要有 $0.01
   - 余额不足时报错文案是 `insufficient_credit` 而不是 `Request Not Allowed`，但顺手查一下

如果以上都正常仍报 403，多半是 Anthropic 风控误伤当前节点 IP，**换个代理节点**试一下。
